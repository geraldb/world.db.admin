# world.db.admin

Open World Database Web Admin Tool in Ruby on Rails (version 3.2 and up)


## Demo

Try the `world.db` Web Admin demo running
on Heroku [`worlddb.herokuapp.com`](http://worlddb.herokuapp.com).
